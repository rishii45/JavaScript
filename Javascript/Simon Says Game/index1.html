How to Play game?
In 'Simon Memory Challenge,' you need to remember a series of buttons and press them in the right order.

It starts easy: In Level 1, if 'A' lights up, press 'A.' In Level 2, if 'B' lights up, first press 'A,' then 'B,' in the order they appeared.

This rule keeps going. In each level, a new button lights up, and you must remember and press all the buttons in the order they lit up. Make one mistake, and it's 'Game Over'.
